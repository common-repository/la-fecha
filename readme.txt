== Description ==

Translates the dates output by your wordpress installation without having to translate everything.

This way you can use an english WordPress install with a spanish theme, and the dates will match.

Please note that this plugin is merely a translation of the original DutchDate plugin, created by Barry Ceelen. More info at https://wordpress.org/extend/plugins/dutchdate/

== Installation ==

1. Copy files to plugin directory and activate.

2. Use date tags in your theme normally, they will be in spanish.
